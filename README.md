## üß† Current Functionality
This prototype acts as a basic logic engine for a car detailing business:
- It categorizes vehicles based on brand and production year.
- It uses .lower() data normalization to ensure user input is processed correctly.
- It predicts the most likely treatment request (Luxury, High-End, or Minimal).
- The predictions are based on client priorities, a client with a car made by a luxurious brand but with a very old production year that isnt a collectors item usually prioritizes the name over the quality. Therefor it predicts the same treatment as a client with a not so luxurious car and a slightly older model.
